# E-commerce Sales Analysis / Data Analytics

This project focuses on analyzing an **E-commerce sales dataset** (sourced from Kaggle) using **Excel, SQL, Python, and Power BI**.  
The objective was to uncover patterns in sales, customer behavior, product performance, and profitability to support **data-driven business decisions**.

---

## Project Overview

The analysis aimed to provide insights into:

- Which products drive the most revenue  
- Who the most valuable customers are  
- Which payment and marketing strategies are effective  
- How profits vary across time and product categories  

Without clear insights, decision-making becomes guesswork, leading to missed growth opportunities and ineffective strategies.

---

## Dashboard Goal

The **Power BI dashboard** was designed to:

- Provide key KPIs: Revenue, Orders, Profit, Customer Insights  
- Visualize sales and profit trends over time  
- Highlight best-performing products & categories  
- Compare payment methods and customer preferences  
- Support strategic decision-making for improving sales, marketing, and customer retention  

---

## Key Features

- **Data Cleaning & Processing:** Done in SQL & Python  
- **Exploratory Data Analysis (EDA):** To uncover meaningful insights  
- **Interactive Dashboard in Power BI:** For visualization and reporting  
- **KPIs Used:** Revenue, Orders, Average Purchase, Profit  

---

## Tools Used

| Tool      | Purpose |
|-----------|---------|
| Excel     | Initial exploration and basic analysis |
| SQL       | Data querying, cleaning, and KPI calculations |
| Python    | Exploratory Data Analysis (EDA), statistical insights, visualizations |
| Power BI  | Dashboard building, storytelling with visuals |

---

## Visuals in the Dashboard

| Visual | Description |
|--------|-------------|
| Revenue by Age (Line Chart) | Shows revenue trends by customer age |
| Product-wise Profit/Loss (Bar Chart) | Highlights best and worst-performing products |
| Payment Method Distribution (Bar Chart) | Shows which payment methods are most used |
| Customer Segmentation by Spending (Donut Chart) | Categorizes customers based on spending behavior |
| Overall KPIs (Cards) | Displays total revenue, orders, profit, and average purchase |

---

## Dataset

The dataset was imported from Kaggle and contains sales transaction records including:

- Order details  
- Customer IDs  
- Product information  
- Units sold, discounts, and profit values  

---

## Outcome

The analysis provides **clear insights into customer behavior, product performance, and sales trends**.  
The dashboard enables stakeholders to make **informed, data-backed business decisions**.
